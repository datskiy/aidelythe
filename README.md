# Aidelythe

A .NET application to organize, manage and coordinate volunteering and community events.


## Status

This project is currently in **early development**. 
Features and structure may change frequently.


## Tech Stack

- .NET 10
- ASP.NET Core Web API
- MediatR
- FluentValidation
- Scalar
- Serilog
- Mapperly
- OneOf
- xUnit
- NSubstitute


## Folder Structure

```

├── src/                 # Application source code
```
